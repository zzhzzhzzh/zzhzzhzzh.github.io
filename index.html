K Nearest Neighbor

本人的第一篇正式博文，这里介绍一下今天刚学习的 stanford cs231n Lecture2 关于knn学习算法的一些总结。

##image classification 的主要方法
形成一个calssifier的主要步骤有三：

  1. 收集image及对应的labels
  2. 通过收集的image训练一个classifier
  3. 使用新image评估classifier

因此我们的knn算法主要有两个函数：

  1. def trans(images,labels)
    记忆所有的数据，不需要做任何处理，正是因为这一点所以knn的training速度十分快但test速度十分慢，而其他学习算法比如cnn等等刚好相反

  2. predict()
    通过最相似的training image预测labels

##如何compare images
上面我们讲到，predict的过程需要找到最相似的图片，knn算法主要通过L1 distance()以及L2 distance()这两个算法来对两张图片进行比较大多数情况下使用L1 distance算法

##训练过程详解
知道如何比较图片之后，接下来就可以训练一个classifier我们首先要对数据进行分类，我们的数据主要分为三类，分别为： training set, validation set and test set

在这之前还要提一个很重要的概念“hyperparameter”， hyperparameter是算法的某些参数，这些参数的值需要在learning process之前被设置好

整个训练过程主要分为三步：

  1. 通过traning set训练赋予不同hyperparameter的算法
  2. 使用validation set选择表现最好的hyperparameter
  3. 对test set仅仅运行一次

validation set 和training set的不同：
算法可以看到training set的label值但是看不到validation set的label值， 算法使用validation set check how well our algorithm is doing.

##knn算法的总结

  1. 我们开始使用带有image和label的training set做训练，然后使用validation set选择最合适的hyperparameter
  2. knn通过nearest training example做预测
  3. 对于knn来说，distance metric and k are hyperparameters
  4. only run on the test set once at the very end!
